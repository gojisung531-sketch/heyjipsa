# Receipt Scanner

영수증 사진 → 정리된 표 + 엑셀 파일.

Claude의 비전으로 영수증을 읽고, 품목별로 카테고리까지 자동 분류해서 마크다운 표와 xlsx로 떨궈주는 Cowork 스킬입니다.

---

## 만든 이유

가계부 입력하기 귀찮아서 만들었습니다.

영수증 사진은 찍어두는데, 그걸 다시 열어서 한 줄씩 옮겨 적는 게 너무 노동이었어요. 편의점/마트/카페 다녀와서 영수증이 5장씩 쌓이면 그냥 포기하고 버렸음.

기존 가계부 앱들은:
- 영수증 OCR이 영어/숫자만 잘 됨 (한국어 품목명에서 자주 깨짐)
- 카테고리 분류가 전부 수동
- 엑셀로 빼려면 별도 결제

근데 Claude는 한국어 영수증도 잘 읽고, "삼각김밥은 식료품, 종량제봉투는 생필품" 같은 분류도 알아서 잘 함. 그러면 그냥 스킬로 만들어서 사진만 던지면 끝나게 하면 되잖아 → 이거 만들게 된 이유.

---

## 어떻게 쓰나요

1. Cowork에 영수증 사진을 첨부합니다.
2. "영수증 정리해줘" 또는 "이거 엑셀로 만들어줘" 라고 말합니다.
3. 끝.

여러 장이면 한꺼번에 첨부해도 되고, 며칠에 걸쳐 하나씩 추가해도 같은 `receipts.xlsx`에 누적됩니다.

### 실행 결과

- 채팅창에 영수증별 마크다운 표
- 카테고리별 소계
- 워크스페이스에 `receipts.xlsx` 파일

---

## 추출되는 항목

| 항목 | 설명 |
|---|---|
| 날짜 | YYYY-MM-DD |
| 매장명 | 영수증 상단 사업자명 |
| 품목명 | 영수증 표기 그대로 |
| 수량 | 정수 |
| 단가 | 원 단위 정수 |
| 합계 | 원 단위 정수 (할인 반영) |
| 카테고리 | 식료품 / 생필품 / 생활용품 / 기타 |

---

## 카테고리 기준

- **식료품**: 먹는 거 전반 (신선식품, 가공식품, 음료, 주류, 과자, 즉석식품, 외식)
- **생필품**: 한 번 쓰면 없어지는 소모품 (휴지, 세제, 샴푸, 치약, 기저귀)
- **생활용품**: 한 번 사면 오래 쓰는 물건 (주방용품, 문구, 의류, 잡화, 전자제품)
- **기타**: 위에 안 들어가는 것 (담배, 복권, 서비스료)

분류가 마음에 안 들면 xlsx에서 직접 고치면 됩니다. 카테고리 컬럼은 자유 편집.

## 한계 (솔직하게)

- 손글씨 영수증, 너무 바랜 감열지는 정확도 떨어집니다.
- 한국 영수증 기준입니다. 해외 영수증은 카테고리 분류가 부정확할 수 있어요.
- OCR은 Claude의 비전 능력에 의존합니다. 별도 OCR 엔진 안 씁니다. 즉, Claude 모델이 못 읽으면 못 읽는 겁니다.
- 카테고리 분류는 100% 정확하지 않습니다. 헷갈리는 품목(예: 다이소 잡화)은 직접 수정하세요.
- 영수증이 아닌 이미지를 던지면 처리 안 됩니다.

## 사용 예시

### 입력
편의점 영수증 사진 1장 + "이거 정리해줘"

### 출력 (채팅)

```
## GS25 강남역점 (2026-05-06)

| 품목 | 수량 | 단가 | 합계 | 카테고리 |
|---|---|---|---|---|
| 삼각김밥 참치마요 | 2 | 1,500 | 3,000 | 식료품 |
| 포카리스웨트 620ml | 1 | 2,400 | 2,400 | 식료품 |
| 종량제봉투 20L | 1 | 580 | 580 | 생필품 |

총액: 5,980원
```

+ `receipts.xlsx` 파일이 워크스페이스에 생성/업데이트됩니다.
