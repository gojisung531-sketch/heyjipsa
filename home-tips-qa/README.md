# home-tips-qa

> 한국 살림 팁을 카테고리별로 모아두고, 질문하면 찾아주고, 새로 들은 팁은 그대로 저장하는 Cowork 스킬

청소·빨래·요리·수납정리·생활꿀팁·절약 6개 카테고리에 기본 팁 60개가 들어있고,
"이거 좋더라" 한 마디만 던지면 알아서 DB에 쌓입니다.

---

## 만든 이유

엄마가 알려준 살림 팁, 유튜브에서 본 청소 팁, 인스타에서 스크롤하다 본 절약 꿀팁 —
필요할 때마다 "그거 뭐였더라" 하면서 다시 검색하게 됩니다.

캡처해두자니 사진 앨범이 쓰레기통이 되고,
메모장에 적자니 카테고리 정리가 안 되고,
노션에 정리하자니 그것 자체가 일이 됩니다.

그래서 그냥 Claude한테 물어보고, Claude한테 알려주면 알아서 쌓이는 구조로 만들었습니다.
"수건 쉰내 어떻게 빼?"라고 물으면 답해주고,
"이거 좋더라: 후라이팬에 밀가루 뿌리면 기름 잘 빠짐"이라고 하면 자동으로 저장됩니다.

---

## 설치

### 옵션 1: Cowork 스킬로 직접 사용

1. 이 레포를 클론하거나 다운로드
2. `home-tips-qa` 폴더 통째로 Cowork 스킬 디렉토리에 복사
3. 끝. Cowork이 알아서 로드합니다.

### 옵션 2: 다른 곳에서 가져다 쓰기

`scripts/search_tips.py`와 `scripts/add_tip.py`는 Python 표준 라이브러리만 사용해서,
어디서든 그대로 import해서 쓸 수 있습니다.

## 사용 예시

### 질문하기

```
나: 수건에서 쉰내 나는데 어떻게 해?

Claude: [빨래] 수건 쉰내 제거
- 내용: 60도 이상 뜨거운 물에 베이킹소다 2큰술 + 식초 1컵 넣고 30분 담근 후 세탁
- 출처: (DB 기본)

💡 세탁기 통세척이 안 되어 있으면 수건이 아니라 세탁기 문제일 수도 있어요.
```

### 새 팁 알려주기

```
나: 이거 좋더라 — 전자레인지에 물+레몬 2분 돌리면 냄새 다 빠짐

Claude: ✅ 팁 저장 완료
카테고리: 청소
팁: 전자레인지에 물+레몬 2분 돌리면 냄새 다 빠짐
출처: (없음)
```

### 카테고리별 모아보기

```
나: 절약 팁 3개만 보여줘

Claude: [절약] 전기세 줄이기 — 멀티탭 스위치
        [절약] 통신비 줄이기
        [절약] OTT 구독료 절약
        ...
```

### 출처 같이 저장

```
나: 유튜브에서 본 건데, 변기에 식초 자기 전 부어두면 다음날 깨끗해짐. https://youtu.be/xxx

Claude: ✅ 저장 완료 (출처 URL 포함)
```

## 구조

```
home-tips-qa/
├── SKILL.md           # Cowork 스킬 정의 (description + 사용법)
├── README.md          # 이 파일
├── tips_db.md         # 팁 DB (기본 60개 포함, 추가될 때마다 누적)
└── scripts/
    ├── search_tips.py # 키워드/카테고리 검색
    └── add_tip.py     # 새 팁 추가 (중복 검사 포함)
```

### tips_db.md 형식

각 팁은 마크다운 헤더로 구분됩니다. 손으로 직접 편집해도 됩니다.

```markdown
## [청소] 전자레인지 내부 냄새 제거
- tip: 머그컵에 물 1컵 + 레몬 반 개 짜서 넣고 2~3분 돌린 후 행주로 닦기
- source: (있으면 URL이나 자유 텍스트)
- added: 2026-05-06
```

## CLI 직접 사용

스킬 없이 터미널에서 바로 쓸 수도 있습니다.

```bash
# 검색
python scripts/search_tips.py "수건 냄새"
python scripts/search_tips.py --category 절약 --limit 5

# 추가
python scripts/add_tip.py --category 청소 --tip "전자레인지에 레몬 돌리면 냄새 제거됨"
python scripts/add_tip.py --category 빨래 --tip "..." --source "https://..."
```

## 카테고리 6개

| 카테고리 | 어떤 팁 |
|---------|--------|
| 청소 | 주방·욕실·가전 청소법 |
| 빨래 | 옷·수건 세탁, 얼룩 제거 |
| 요리 | 보관법, 조리 꿀팁, 시간 단축 |
| 수납정리 | 공간 활용, 정리 시스템 |
| 생활꿀팁 | 위 카테고리에 안 맞는 잡다한 것 |
| 절약 | 전기·가스·수도·통신·식비 |

새 카테고리를 추가하고 싶으면 `scripts/search_tips.py`와 `scripts/add_tip.py`의 `CATEGORIES` 리스트에 추가하면 됩니다.

## 한계

- 키워드 매칭 기반이라 "운동화 누렁이"와 "운동화 황변"은 다른 검색으로 인식됩니다.
  더 정확한 검색이 필요하면 형태소 분석기(KoNLPy 등)를 추가하세요.
- 외부 URL 유효성 검증 안 함. 사용자가 준 URL을 그대로 저장합니다.
- 중복 검사는 Jaccard 유사도(80% 임계값) 기반이라, 표현이 많이 다르면 중복을 못 잡을 수 있습니다.
