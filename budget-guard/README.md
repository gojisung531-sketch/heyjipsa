# Budget Guard 🛒

장바구니 붙여넣으면 사치품 골라내고 충동구매 막아주는 Claude 스킬.

## 만든 이유

쇼핑 카트에 일단 담아놓고 결제 직전에 "이거 다 진짜 필요한가?" 후회하는 패턴이 너무 많았다.

- 휴지 같은 생필품이랑 마카롱 같은 사치품이 같은 카트에 섞여 있으면 죄책감이 무뎌진다.
- "이번 달에 마카롱 3번째 사는 중"이라는 사실을 본인은 모른다.
- 월 예산은 머릿속에만 있고, 결제 순간에 누가 알려주는 사람이 없다.

그래서 만들었다. Claude가 카트 보고 ✅ 생필품 / ⚠️ 준생필품 / 🚨 사치품 딱 분류해주고, 같은 거 또 사면 "이번 달 3번째예요" 짚어주고, 예산 80% 넘으면 경고 띄움.

## 뭐 하는 스킬인지

- 장바구니 텍스트 붙여넣기 → 자동 분류 + 가격 합계
- 사치품 항목에 🚨 + 한 줄 경고
- 같은 품목 반복 구매 감지 (`"X 이번 달 N번째 구매예요"`)
- 월 예산 입력하면 80% / 100% 초과 경고
- 카테고리별 비율 막대 그래프

## 사용 예시

입력:

```
휴지 24롤 18,000원
샤넬 립스틱 52,000원
라면 5개 4,500원
마카롱 6개 15,000원
우유 1L 3,200원
```

출력 (요약):

```
| 품목 | 가격 | 분류 |
|------|------|------|
| 휴지 24롤 | 18,000원 | ✅ 생필품 |
| 샤넬 립스틱 | 52,000원 | 🚨 사치품 |
| 라면 5개 | 4,500원 | ✅ 생필품 |
| 마카롱 6개 | 15,000원 | 🚨 사치품 |
| 우유 1L | 3,200원 | ✅ 생필품 |

✅ 생필품: 3개 / 25,700원 (28%) █████
🚨 사치품: 2개 / 67,000원 (72%) ██████████████

🚨 사치품 경고
- 샤넬 립스틱 (52,000원) — 정말 필요한가요?
- 마카롱 6개 (15,000원) — 정말 필요한가요?
```


## 직접 실행

CLI:

```bash
python classifier.py < cart.txt
```

함수 직접 호출:

```python
from classifier import analyze_cart, format_report

result = analyze_cart(
    text=cart_text,
    monthly_budget=300000,
    current_spent=180000,
    history={"마카롱 6개": 2},  # 이번 달 이미 2번 샀음
)
print(format_report(result))
```

## 카테고리 커스터마이징

`classifier.py` 의 `CATEGORIES` 딕셔너리에 키워드만 추가하면 된다.

```python
CATEGORIES["사치품"]["keywords"].append("내가 자꾸 사는 그 브랜드")
```
