# voice-to-todo

> 음성으로 떠오르는 대로 말하면 자동으로 카테고리별 할 일 리스트가 되는 Cowork 스킬

## 만든 이유

머릿속에 할 일이 우후죽순 떠오를 때마다 "장보기 / 집안일 / 업무 / 경조사" 카테고리 정해서 입력하는 게 너무 귀찮았다.
그냥 떠오르는 대로 다 말해버리고, **분류는 AI가 하면 되잖아?** 라는 생각으로 만들었다.

휴대폰 음성 받아쓰기 → Claude에 붙여넣기 → 자동으로 정리된 TODO.md.
이게 이 스킬이 하는 일의 전부다.

## 어떻게 동작하나

1. 자연어 문장에서 줄/문장 단위로 할 일을 분리
2. 키워드 기반으로 카테고리 자동 분류 (장보기 / 집안일 / 업무 / 경조사 / 기타)
3. 우선순위 추정 (`급`, `오늘`, `마감` 같은 단어 → 상)
4. 기한 추출 (`금요일`, `내일`, `5월 10일` 같은 표현)
5. 기존 `TODO.md`에 append, 카테고리별로 정렬해서 다시 저장

## 사용 예시

### 입력

```
화장실 휴지 사야 돼
금요일까지 보고서 제출해야 함
엄마 생신 선물 준비
```

### 출력

**새로 추가된 항목**

| 카테고리 | 항목 | 우선순위 | 기한 |
|---------|------|---------|------|
| 장보기 | 화장실 휴지 | 중 | - |
| 업무 | 보고서 제출 | 상 | 금요일 |
| 경조사 | 생신 선물 준비 | 중 | - |

**전체 리스트(TODO.md)**

```markdown
# 할 일 리스트

## 업무
- [ ] 보고서 제출 (상, ~금요일)

## 장보기
- [ ] 화장실 휴지 (중)

## 경조사
- [ ] 생신 선물 준비 (중)
```

## 설치

이 스킬은 Cowork 스킬 형식이라 별도 설치 없이 폴더만 있으면 된다.
Claude에게 폴더를 마운트하고 "이 폴더에 있는 voice-to-todo 스킬 써줘" 하면 끝.

직접 CLI로 돌리고 싶으면:

```bash
cd voice-to-todo
python add_todo.py "화장실 휴지 사야 돼, 금요일까지 보고서 제출"
```

옵션:

- `--file input.txt` : 파일에서 텍스트 읽기
- `--todo-path ./my_todo.md` : 저장 위치 지정 (기본 `./TODO.md`)
- `--dry-run` : 파일 안 쓰고 출력만

## 파일 구조

```
voice-to-todo/
├── SKILL.md          # Claude가 트리거 판단할 때 읽는 명세
├── README.md         # 이 파일
├── parser.py         # 자연어 → Todo 파싱 코어
└── add_todo.py       # CLI 진입점, TODO.md 읽기/쓰기
```

## 한계

- 한국어 전용. 영어 섞이면 "기타"로 빠질 가능성 높음.

- 음성 인식(STT)은 포함 안 됨. 받아쓴 텍스트만 처리.
- 우선순위·기한은 휴리스틱 추정. 100% 정확 보장 X.
- 외부 API 안 씀 → 차단 걱정은 없음.
