# Voice to Todo

## What this skill does

자연어로 말한 할 일을 자동 파싱해서 카테고리/우선순위/기한을 뽑아내고, 마크다운 할 일 리스트(TODO.md)에 append 한다. 한 번에 여러 줄 입력 가능.

## When to use

- 사용자가 "할 일 추가해줘", "투두에 넣어줘", "받아 적어줘" 같은 표현을 쓸 때
- 사용자가 음성으로 받아쓴 텍스트(여러 할 일이 한 덩어리로)를 던질 때
- "~사야 돼", "~해야 해", "~까지 제출", "~예약해야 함" 같은 할 일 표현이 1개 이상 등장할 때
- 기존 TODO.md 파일에 새 항목을 추가하고 싶을 때

## How to use

1. 사용자 입력에서 할 일 후보 문장들을 줄 단위 또는 문장부호 기준으로 분리한다.
2. 각 문장을 `parser.py`의 `parse_todo()` 에 넣어 다음을 추출한다:
   - 카테고리: 장보기 / 집안일 / 업무 / 경조사 / 기타
   - 항목명 (핵심 명사구)
   - 우선순위: 상 / 중 / 하 (기본값 중)
   - 기한 (없으면 빈 값)
3. 작업 폴더에 `TODO.md`가 있으면 읽고, 없으면 새로 만든다.
4. 새 항목을 append 하고 카테고리별로 재정렬한다.
5. 결과를 마크다운 테이블 + 카테고리별 그룹으로 보여준다.

## Output format

두 가지를 한 번에 출력한다.

**1) 새로 추가된 항목 테이블**

| 카테고리 | 항목 | 우선순위 | 기한 |
|---------|------|---------|------|
| 장보기 | 휴지 | 중 | - |
| 업무 | 보고서 제출 | 상 | 금요일 |

**2) 전체 리스트(카테고리별 정리)**

```
## 장보기
- [ ] 휴지 (중)
- [ ] 우유 (중)

## 업무
- [ ] 보고서 제출 (상, ~금요일)
```

마지막에 TODO.md 파일 경로를 computer:// 링크로 제공한다.

## Examples

**예시 1 - 한 줄 입력**

입력: "화장실 휴지 사야 돼"
출력:
| 카테고리 | 항목 | 우선순위 | 기한 |
|---------|------|---------|------|
| 장보기 | 화장실 휴지 | 중 | - |

**예시 2 - 기한 포함**

입력: "금요일까지 보고서 제출해야 함"
출력:
| 카테고리 | 항목 | 우선순위 | 기한 |
|---------|------|---------|------|
| 업무 | 보고서 제출 | 상 | 금요일 |

**예시 3 - 여러 줄 한 번에 (음성 받아쓰기 시나리오)**

입력:
"엄마 생신 선물 준비해야 하고, 화장실 청소도 해야 함. 그리고 우유 사와야 돼"

출력:
| 카테고리 | 항목 | 우선순위 | 기한 |
|---------|------|---------|------|
| 경조사 | 생신 선물 준비 | 중 | - |
| 집안일 | 화장실 청소 | 중 | - |
| 장보기 | 우유 | 중 | - |

## Edge cases

- **빈 입력**: "추가할 할 일을 알려주세요" 안내 후 종료.
- **카테고리 애매**: 키워드 매칭 안 되면 "기타"로 분류. 사용자에게 재분류 원하는지 묻지 않음 — 그냥 기타로 넣고 끝.
- **너무 긴 입력 (>2000자)**: 그대로 처리 가능. 줄/문장부호 단위로 잘라서 각각 파싱.
- **중복 항목**: 기존 TODO.md에 동일 항목명이 있으면 추가하지 않고 "이미 있음" 표시.
- **TODO.md 손상/잘못된 형식**: 백업 파일(TODO.bak.md)로 저장 후 새로 만든다.

## Limitations

- 한국어 휴리스틱 기반 분류기. 키워드 매칭이라 완벽하지 않음.
- 기한 파싱은 요일(월~일), "오늘/내일/모레/다음주", "MM월 DD일" 정도만 인식. 복잡한 날짜 표현은 원문 그대로 저장.
- 음성 인식(STT) 자체는 포함하지 않음. 받아쓴 텍스트 입력만 처리.
- 우선순위는 키워드("급", "빨리", "오늘", "마감") 기반 추정. 정확도 보장 안 함.
